Valgrind安装与memcheck简单使用

安装 valgrind

sudo apt install valgrind

安装后，查看一下版本以确认安装成功

valgrind --version

输出形如以下版本号信息就表示安装成功

使用 memcheck 检查内存泄漏

memleak.cpp

其中 --tool=memcheck 可以省略，因为 valgrind 的默认工具就是 memcheck 。
